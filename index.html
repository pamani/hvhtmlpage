Frequently Asked Questions

There are many questions that may come to one's mind when they see this website. All those questions are answered here.

  • For What purpose was this website created?

    This website was created as a project to be uploaded in github for a program named Future Ready Talent. This website was supposed to use one or more azure services and fall under the category health as I selected that topic.

  • What is the main topic of attention in the website?

    The main topic of attention in this website is the chatbot that is supposed to interact with the patients who will use this kind of website. The names and pictures used in this project is all for eductional purposes and the resemblence of the names in the project with the real world people is purely coincidental.

  • What does the chatbot tell the patients about?

    The chatbot tells the patients about the information about diseases, the doctors of the hospital, info about the hospital and since it is the era of covid-19 the bot also tells the patients about covid-19 and its precautions. There are a lot of diseases and it is a mini project so there are possibilities that the chatbot might not answer to each and every questions that a patient asks.

  • How can we creat such a chatbot for ourself and integrate it with our own website?

    With Azure Student Pack or though an actual azure subscription anyone can create a chatbot through creating a Question and Answer Knowledgebase and then integrating it to a web chatbot which can then be put in any website using iframe.(i.e If you know HTML and CSS its pretty easy.)

  • Can the chatbot steal my identity or save my conversations?

    Whatever the user askes is answered only through checking the already synthesised Knowledgebase and no other data from the user is saved anywhere for any purposes. Please avoid uploading your documents in the attachment of the chatbot. It accepts attachments but it only replies to text based messages and not to documents or pictures!
